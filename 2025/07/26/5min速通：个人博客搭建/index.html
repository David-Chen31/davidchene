5min速通：个人博客搭建

非常好！你想要：

✅ 能修改主题
✅ 上线运行，不依赖本地环境

那么我强烈推荐你使用 Hexo + GitHub Pages。它功能强、主题丰富、可完全上线部署，无需服务器，甚至不用购买域名也能运行！

  3. 注册 GitHub 账号：github.com
  4. 创建 GitHub 仓库：

    • 不要勾选 README
